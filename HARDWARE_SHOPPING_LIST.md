## 📝 Component Specifications Reference

### SIM808 Module Power Requirements
```
Operating Voltage: 3.4V - 4.4V (typically 3.7-4.2V)
Typical Consumption: 100mA @ 3.7V
Peak Current (GSM transmit): 2A @ 3.7V
Peak Current (GPS): 30mA @ 3.7V
```

### Arduino Uno Power Requirements
```
Operating Voltage: 7-12V DC (via barrel jack) or 5V USB
Typical Consumption: 50mA (without peripherals)
Maximum I/O Current per Pin: 40mA
Maximum Total Current (all pins): 200mA
```

### Buzzer Specifications
```
Operating Voltage: 3-5V DC
Current Draw: 20-30mA
Sound Level: 80-90dB @ 10cm
Frequency: 2000-4000Hz (typical)
```

## ✅ Pre-Purchase Checklist

Before buying, verify:

**SIM808 Module:**
- [ ] Comes with GPS antenna
- [ ] Comes with GSM antenna
- [ ] Has IPEX/U.FL connectors for antennas
- [ ] Board is genuine (not clone with issues)
- [ ] Includes documentation or datasheet

**Power Supply:**
- [ ] Voltage matches requirement (5V for SIM808)
- [ ] Current rating adequate (minimum 2A)
- [ ] Has correct connector or terminals
- [ ] Includes power cable

**SIM Card:**
- [ ] PIN lock is disabled
- [ ] Has data bundle active
- [ ] Has airtime for SMS
- [ ] Compatible with SIM808 (standard size or adapter)

**Enclosure (for field deployment):**
- [ ] Large enough for all components
- [ ] Rated IP65 or better
- [ ] Has mounting holes or options
- [ ] Allows antenna cables to exit

## 💡 Money-Saving Tips

1. **Bulk Buying**: If building multiple trackers, buy components in bulk
2. **Local Sourcing**: Check local electronics/computer shops first
3. **Alternative Components**: 
   - Old phone charger for 5V power (if rated 2A+)
   - Plastic food container as temporary enclosure
   - Repurpose old GPS/GSM antennas
4. **DIY Enclosure**: Build wooden/metal box instead of buying
5. **Battery Recycling**: Use old UPS or car battery (test health first)
